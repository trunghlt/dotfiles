Vi-improved:
===========

## Vim's shortcuts:

- F2: save in normal/insert mode
- F3: open a file in normal mode
- `-F3: open a file in new tab in normal mode
- F4: open NERD tree
- Ctrl-f: delete a word after cursor in insert mode
- Ctrl-e: exit
